# PromptBreak demo narration

Generative AI saves time, but every run creates an awkward waiting pocket. PromptBreak turns that dead time into a quick, playable debugging break.

Choose an original Patchling, pick a software-themed mission, and move while attacks fire automatically. Each short run adds upgrades such as Force Quit Wave, Power Patch, and wider attack scope.

The progress bar turns the build into a story: clear bugs, survive critical phases, and defeat two bosses before the task completes.

When the Patch Parade is charged, all nine Patchlings arrive and perform their signature attacks together.

I used Codex with GPT-5.6 to iterate on gameplay, animation, responsive UI, security checks, and the original character system. The future desktop companion can also pause the run when an AI task finishes.

PromptBreak makes waiting feel intentional: AI works, you play, then you return refreshed.
